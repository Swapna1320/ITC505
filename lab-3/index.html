Delivering Fonts to a Client Browser

Using a web font provider like Google Fonts is one of the easiest methods to provide a client with a custom font. We just include a link to the font in our HTML document, which is hosted by Google. This makes sure the user doesn't need to have the font installed on their computer because it loads dynamically when the webpage is viewed.

Steps to Implement

  1. I choosed a font from Google Fonts.
  2. Include the provided link inside the <head> section of my HTML document.
  3. And applied the font to elements using CSS.

For this page I went with 'Fruktur' font, which has a funky, playful appearance. This font is unlikely to be pre-installed on most devices, making it an ideal choice for this lab.
